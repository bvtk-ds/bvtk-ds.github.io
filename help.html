BVTK Help Page

BVTK is a more simplified and larger view of Trello boards, broken into an organized task board inspired by Kanban boards.
Below is a guide on how to use BVTK within Trello as well as ways to improve your experience with it via parameters.


Using Trello with BVTK

BVTK takes advantage of the label system in Trello. It's based off of the color of the label and not the text inside it. It looks from the first to last label to determine which column to place the card in.

Backlog tasks need a Black colored label.
To Do tasks need a Yellow colored label.
In Progress tasks need a Cyan colored label.
Complete tasks need a Green colored label.

Parameters

Currently, in order to improve your use of BVTK, you'll have to manually add parameters to the URI.
Below are all of the current parameters you can use.

columns

This is a way to reduce which columns show up on screen. It goes in order from left to right, Backlog being your 0 column and Complete being your 3 column. You can exclude multiple columns at once by using "," as a separater.
  • Removing Backlog
    • https://bvtk-ds.github.io/?columns=0
  • Removing Complete
    • https://bvtk-ds.github.io/?columns=3
  • Removing Backlog and Complete
    • https://bvtk-ds.github.io/?columns=0,3

exclude

Exclude is an Trello Board exclusion filter. By default, every kind of board is include except for Organization owned boards. You can exclude multiple boards at once by using "," as a separater.
  • Removing the Big View Trello Kanban board
    • https://bvtk-ds.github.io/?exclude=Big View Trello Kanban

ts

Adjusts the text size of the task cards only. The size is evaluated as *em* values and defaults to 1 em.
  • Default size enforced
    • https://bvtk-ds-github.io/?ts=1
  • Upping to 1.25em
    • https://bvtk-ds-github.io/?ts=1.25
